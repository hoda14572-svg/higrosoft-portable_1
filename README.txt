HigroSoft – Portable EXE (Electron) – Build fără instalări pe PC-ul tău
============================================================================
Ținta: un singur fișier HigroSoft.exe (portabil, fără instalare, 100% offline).

Cum obții EXE-ul fără să instalezi nimic local:
1) Creează un repo gol pe GitHub (ex: higrosoft-portable)
2) Urcă conținutul acestui folder (inclusiv .github/workflows/build-windows.yml)
3) Accesează tab-ul Actions și pornește workflow-ul „Build HigroSoft Portable EXE”
4) După execuție, descarcă artifactul „HigroSoft_Portable_EXE”
5) Vei avea fișierul: HigroSoft.exe (portabil, dublu-click și rulează)

Note:
- EXE-ul include Chromium ⇒ ~100–120 MB
- Înlocuiește icon-ul app/assets/higrosoft.ico cu varianta ta (256x256)
- Aplicația funcționează 100% offline, datele nu părăsesc calculatorul.
